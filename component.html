🌱 Smart Hydroponics System Components

Arduino နဲ့တည်ဆောက်ထားတဲ့ Smart Hydroponics System မှာ အသုံးပြုထားတဲ့ sensors, modules နဲ့ hardware များ

🤖 Arduino Uno

Main controller board for system automation and sensor data processing.

📏 Ultrasonic Sensor

Water level ကို အတိအကျတိုင်းတာပေးသည်။

⚡ EC Sensor

Nutrient concentration (Electrical Conductivity) ကိုတိုင်းတာသည်။

🧪 pH Sensor (PH-4502C / SEN0161)

Water acidity / alkalinity ကို စောင့်ကြည့်ပေးသည်။

🌡️ DHT11 / DHT22

Temperature & Humidity monitoring system.

💡 LDR Sensor

Light intensity ကို detect လုပ်ပေးသည်။

💧 Water Pump

Plant system အတွက် water circulation လုပ်ပေးသည်။

🔌 Relay Module

Pump & fan တွေကို automatic control လုပ်ရန်အသုံးပြုသည်။

🖥️ LCD 16x2 (I2C)

Real-time sensor data ပြသရန်အသုံးပြုသည်။

❄️ DS18B20 Waterproof

Water temperature measurement sensor ဖြစ်သည်။

🔋 Power Supply

12V 2A / 5A Adapter + DC Jack Connector

🧱 Structure

PVC pipes, tubes, breadboard, jumper wires
