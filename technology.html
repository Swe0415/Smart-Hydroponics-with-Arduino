TECH STACK

Technologies Used

This project combines modern web development technologies with Arduino automation systems to create a smart hydroponics farming solution.

Website Development

🌐

HTML5

Used to build the structure and content layout of the website.

🎨

CSS3

Used for responsive design, animations, glass UI, and modern styling.

⚡

JavaScript

Adds interactivity such as animations, navbar effects, and dynamic UI behavior.

🖥️

PHP

Used for backend development, handling forms, and connecting with databases.

🗄️

MySQL

Used to store project data, sensor logs, and system information.

☁️

Netlify / Hosting

Used to deploy and host the hydroponics website online.

Arduino & Smart Farming

🔌

Arduino Uno

Main microcontroller used to control all sensors and automation systems.

🖥️

Wokwi Simulation

Used for Arduino testing and circuit simulation.

📡

Blynk IoT Platform

Used for remote monitoring and smart control using mobile devices.
